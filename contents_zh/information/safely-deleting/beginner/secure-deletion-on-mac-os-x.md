Mac OS X安全刪除

在 OS X 10.4或更新的作業環境下, 安全地刪除檔案很容易作到:
* 把檔案移到資源回收筒;
* 然後在"Finder"選擇 >「安全清空垃圾筒」 “Secure Empty Trash” 

或者你想要安全地刪除過去自電腦上已刪除的資料.這種作法蘋果的建議是:
* 開啟「磁磁管理工具」"Disk Utility" (in Applications/Utilities);
* 選擇Help > Disk Utility Help
* 進一步了解如何清除用未的硬磁空間.
